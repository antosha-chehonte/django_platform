Импорт подразделений из CSV

Формат CSV файла:

Разделитель: точка с запятой (;)

Обязательные колонки:

  • Название
  • Код (уникальный)

Опциональные колонки:

  • Код сортировки (может быть пустым)
  • Описание (может быть пустым)
  • Короткое наименование (может быть пустым)
  • Email (может быть пустым, формат: example@domain.com)
  • Почтовый индекс (может быть пустым)
  • Город (может быть пустым)
  • Улица (может быть пустой)
  • Здание (может быть пустым)
  • Идентификатор узла (может быть пустым, максимум 4 символа)
  • IP адрес (может быть пустым, формат: 192.168.1.0)
  • Маска (может быть пустым, значение от 0 до 32)
  • Родительское подразделение (название или код, может быть пустым)
  • Логическое (True/False, по умолчанию: False)
  • Активно (True/False, по умолчанию: True)
