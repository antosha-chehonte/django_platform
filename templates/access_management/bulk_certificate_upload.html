Информация о массовой загрузке

Вы можете загрузить несколько сертификатов одновременно. Система автоматически:

  • Распарсит каждый файл сертификата
  • Извлечет данные (серийный номер, отпечаток, дату окончания)
  • Найдет сотрудника по ФИО из сертификата
  • Создаст записи для найденных сотрудников
  • Переименует файлы по маске: фамилия_имя_отчество__YYYY_MM_DD.расширение

Важно: Сертификаты, для которых сотрудник не найден в базе данных, будут пропущены.
